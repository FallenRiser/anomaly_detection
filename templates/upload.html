📋 Required CSV Format

Your CSV file must include these required columns:

Column Type Required Description
date YYYY-MM-DD ✅ Date of the inventory record
product_id String ✅ Product SKU identifier
stock_level Integer ✅ Current stock quantity
is_anomaly 0/1 ✅ Whether this record is an anomaly
product_name String Optional Human-readable product name
category String Optional Product category
distribution_center_id String Optional DC identifier
distribution_center String Optional DC name
region String Optional Geographic region
sales_rate Float Optional Daily sales rate
demand_forecast Float Optional Predicted demand
reorder_point Integer Optional Reorder threshold
lead_time_days Integer Optional Supplier lead time
anomaly_type String Optional stockout / overstock / demand_spike / supply_delay / shrinkage / forecast_error
anomaly_severity String Optional critical / high / medium

Example:

date,product_id,product_name,stock_level,sales_rate,is_anomaly,anomaly_type,anomaly_severity

2025-06-16,SKU001,Tide Laundry 3L,5,35.0,1,stockout,critical
